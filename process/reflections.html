Reflections in code

To do reflections in code, we simply create a a new ray at the intersection point that has a reflected direction.

Equations for a Reflected ray.

Using the value of d we got from the line-sphere intersection and the equation for a ray, we can find the location of the intersection:
y = o + dl
Where y is the intersection point found before.

The unit vector that points from the center of the sphere to that point, or normal, can be found as follows:
n = (y - c) / ||y - c||

The reflected ray's direction can then be found by:
t = l - 2(n*d)n

The equation for the reflected ray is:
x = y + er
Where e is the distance for the reflected ray.
