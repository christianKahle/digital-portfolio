What is a Vector?

A vector is a set of numbers that can be defined in a list
Ex: [ 10, 2], [ 3, 0, 2]
Or, with a direction and magnitude (length)
Ex: [ 20°, 2], [ 10°, 30°, 3]

You can also do math with vectors, for example:
P = [ 2, 0, 5]
L = [ 3, 1,-7]
P + L = [ 5, 1,-2]


What is a scalar?

A scalar is a number that has only magnitude, which can be represented by a single number
Basically, any 'normal' number.
Ex: 2, 4.

  • Addition and Subtraction ( + - )
  • Scalar Multiplication ( s(v) )
  • Dot Product ( * )
  • Cross Product ( x )
  • Magnitude (|v|)
  • Normalization ( v̂ )
Vector Operations: Adding and Subtracting

a + b = c = [a1,a2,a3...] + [b1,b2,b3...] = [c1,c2,c3] = [a1+b1,a2+b2,a3+b3...]
a - b = c = [a1,a2,a3...] - [b1,b2,b3...] = [c1,c2,c3] = [a1-b1,a2-b2,c1-c2...]
Vector Operations: Scalar Multiplication

Multiplication
s(v1) = s*(v1) = v2
Is equivalent to:
s[a1,b1,c1...] = [s(a1),s(b1),s(c1)...]

Division
Scalar division is accomplished by taking the reciprocal of the divisor and multiplying. (v1)/s = 1/s * (v1)
Vector Operations: Length and Magnitude

|v| = √(v * v) = √(v^2) = √([a1,a2,a3...] * [a1,a2,a3...]) = √(a1^2 + a2^2 + a3^2...)
*This produces a scalar value*

Vector Operations: Normalization

v̂ = v/|v| = 1/|v|(v)
*This produces a unit vector with the direction of the original vector.*

Unit Vectors
These are important.
A unit vector can be denoted by the little hat above the variable.

All unit vectors have a length of one.

Because the length is one, a unit vector specifies only direction.
Vector Operations: Dot Product

a * b = s = [a1,a2,a3...] * [b1,b2,b3...] = a1*b1 + a2*b2 + a3*b3...
 *This produces a scalar value*
Vector Operations: Cross Product

v1 x v2 = v3 = [a1,a2,a3] x [b1,b2,b3] = [c1,c2,c3]
c1 = a2*b3 - a3*b2
c2 = a3*b1 - a1*b3
c3 = a1*b2 - a2*b1
 *This produces a vector at a right angle to the plane made by v1 and v2 according to the right hand rule*
**This can only be done with 3-dimensional and 7-dimensional vectors**
How does it get implemented?

We can implement vectors using a double or float array with [x,y,z].
With this format, directions are represented with unit vectors.
